## 📦 Install
```
npm install kku-ui
```
```
yarn add kku-ui
```

<br>

## ⚙️ Initial Setting
```typescript
// 🏃🏻‍♂️ Entry Point

import 'kku-ui/lib/styles/index.css'
```

<br>


## 🧑🏻‍💻 Use Examples

```typescript
import { KButton } from 'kku-ui'

export default function Component() {
    return (<KButton primary label="Hello World!" />);
} 
```
